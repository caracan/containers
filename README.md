# Description

A set of containers I use for home and toolbox containers on Fedora.

## Containers

| Container Image | Description | Arch |
| --- | --- | --- |
| ansible-toolbox | Toolbox container with Ansible tools | amd64 |
| keepalived | UBI9 container with keepalived | amd64, arm64 |
| keepalived-init | As above but based on the ubi-init container with systemd | amd64, arm64 |
| apd-cli | Toolbox container with with various cli tools | amd64 |
| caddy | Caddy image with Cloudflare plugins for LetsEncypt | amd64 |
